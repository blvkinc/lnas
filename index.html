Angular Dependencies:

  • "@angular/core": The core Angular framework.
  • "@angular/common": Common utilities, directives, and pipes used in Angular applications.
  • "@angular/forms": Support for building forms and managing form inputs.
  • "@angular/router": Routing module for navigating between different components.
  • "rxjs": Reactive Extensions for JavaScript, used for handling asynchronous operations.

Java Spring Boot Dependencies:

  • "spring-boot-starter-web": Web framework for building RESTful APIs.
  • "spring-boot-starter-data-jpa": Support for accessing and managing the MySQL database using Java Persistence API (JPA).
  • "spring-boot-starter-security": Provides security features like authentication and authorization.
  • "spring-boot-starter-jwt": JWT library for handling JSON Web Tokens.
  • "mysql-connector-java": Connector for connecting Java applications to MySQL database.

JWT Dependencies:

  • "jjwt": A Java library for working with JWT tokens, including token generation, validation, and parsing.

MySQL Database Driver:

  • "mysql-connector-java": JDBC driver for connecting to the MySQL database from Java applications.

Spring Boot Dependencies:

  • "spring-boot-starter-web": Web framework for building RESTful APIs.
  • "spring-boot-starter-data-jpa": Support for accessing and managing the database using Java Persistence API (JPA) with Hibernate.
  • "spring-boot-starter-security": Provides security features like authentication and authorization.
  • "spring-boot-starter-test": Dependencies for testing the application.

Database Dependencies:

  • "mariadb-java-client": JDBC driver for connecting to the MariaDB database from Java applications.

Swagger Dependencies:

  • "springdoc-openapi-ui": Provides integration with OpenAPI/Swagger UI for API documentation and exploration.

Logging Dependency:

  • "slf4j-api": Logging facade for various logging frameworks.
